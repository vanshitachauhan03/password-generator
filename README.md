# 🔐 Password Generator (Full Stack)

A full-stack password generator built using Flask and React.

## 🚀 Features
- Generate secure passwords
- Strength indicator
- REST API integration

## 🛠️ Tech Stack
- Python (Flask)
- React.js
- Axios

## 📁 Structure
backend/ → Flask API  
frontend-ui/ → React UI  

## ▶️ Run Locally

### Backend
cd backend  
pip install flask flask-cors  
python run.py  

### Frontend
cd frontend-ui  
npm install  
npm start
